# Deep Learning Research: ResNet vs MLP

Тут будет разобрана немного тема **деградации градиента** в глубоких нейронных сетях. В рамках этого проекта было проведено сравнение архитектур **MLP** и **ResNet** на различных глубинах.

## Основная идея

Хотелось экспериментально подтвердить, почему глубокие архитектуры чаще прибегают к residual блокам. Также хотелось посмотреть, как **Batch Normalization** влияет на процесс обучения в зависимости от места его расположения (**Pre-norm** vs **Post-norm**).

## Данные

Для тестов генерируется синтетический сигнал, сочетающий в себе:
* Гармонические колебания
* **Детерминированный хаос**
* Гауссовский шум.

Я протестировал 5 конфигураций:
1. **MLP** (Pure)
2. **MLP + Post-Norm BN**
3. **ResNet** (Pure)
4. **ResNet + Post-Norm BN**
5. **ResNet + Pre-Norm BN**

Каждая модель обучалась на сетке глубин: `[2, 10, 20, 30, 40]` слоев.
